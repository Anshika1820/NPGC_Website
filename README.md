🎓 NPGC Website

The NPGC Website is a simple, clean, and responsive frontend project built using HTML and CSS. It is designed to present college-related information in a structured and visually appealing manner, serving as the digital face of the institution.

✨ Features

📌 Responsive Layout – Adapts smoothly to desktops, tablets, and mobile devices.
🎨 Modern UI Design – Clean and attractive styling using CSS.
🧭 Navigation Bar – Easy navigation to different sections of the site.
📖 Informative Sections – Includes details about the college, academics, events, and contact information.

⚙️ Tech Stack

HTML5 – For structuring web content

CSS3 – For styling, layout, and responsive design

🚀 How It Works

Open the index.html file in any modern web browser.

Navigate through the different sections using the navigation bar.

Explore college-related information presented in a clean and structured format.
